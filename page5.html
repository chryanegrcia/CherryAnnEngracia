Exercise Reflection

Practicing how to create a daily schedule in HTML has helped me understand the structure and organization of web content more clearly. Through this exercise, I learned how to use basic HTML elements such as tables, headings, and divs to lay out information in a logical and visually accessible format. It also improved my attention to detail, as aligning time slots and activities required precision. By building a daily schedule, I not only practiced coding but also applied real-life planning skills, which made the experience both practical and engaging. Overall, it strengthened my foundational knowledge in HTML and gave me confidence to create more complex web layouts in the future.
