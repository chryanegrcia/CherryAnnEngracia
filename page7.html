Exercise Reflection

When it comes to making an outline in CSS, I've learned that there are three main approaches: inline, internal, and external. Inline CSS involves applying styles directly to an HTML element, which can be useful for small changes but leads to cluttered code. Internal CSS defines styles within the HTML document's head, suitable for small projects but prone to duplication. External CSS, on the other hand, involves creating a separate CSS file, promoting code organization and reusability. In my experience, external CSS is the way to go, as it makes maintenance and updates a breeze.
